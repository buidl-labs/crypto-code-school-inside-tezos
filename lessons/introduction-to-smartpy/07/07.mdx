Entry Point is a method of a contract class that can be called from the outside. They must take a `self` parameter and an optional parameter `params` if the method takes user's input. They need to be marked with the `@sp.entry_point` decorator. For example the following entry point checks that the argument given is greater than 2 or not.

```python
@sp.entry_point
def checkValue(self, params):
   sp.verify(params.x > 2)
```
